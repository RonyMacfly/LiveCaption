
# Installation

Pip:
```
1) sudo apt install portaudio19-dev

2) pip3 install git+https://github.com/RonyMacfly/LiveCaptionLinux

3) export PATH="/home/$USER/.local/bin:$PATH"

4) Audio mixer -> Volume Control -> Recording -> ALSA plug-in -> Monitor of Built-in Audio Analog Stereo

5) LiveCaptionLinux
```
